### 2. Исправлено дублирование критериев ⚡

**Проблема:**
Критерии создавались несколько раз при создании сессии оценки.

**Решение:**
```python
# backend/.../signals.py
if created and not kwargs.get('raw', False):
    # Проверяем, есть ли уже критерии
    existing_count = AssignedCriterion.objects.filter(
        evaluation_session=instance
    ).count()
    
    if existing_count == 0:
        # Создаем все критерии за один раз через bulk_create
        AssignedCriterion.objects.bulk_create(
            assigned_criteria_to_create,
            ignore_conflicts=True  # Игнорируем дубликаты
        )
```

**Улучшения:**
- ✅ Проверка существующих критериев перед созданием
- ✅ `bulk_create` для быстрого создания всех критериев за один запрос
- ✅ `ignore_conflicts=True` для защиты от дубликатов
- ✅ Проверка флага `raw` для импорта данных

### 3. Автоматическое обновление статуса на "Завершено" ✔️

**Проблема:**
После сохранения всех оценок статус сессии оставался "В ожидании".

**Решение:**
```javascript
// frontend/.../EvaluationInput.js
// Проверяем, все ли критерии заполнены
const allFilled = assignedCriteria.every(
  ac => answers[ac.id]?.score_value
);

if (allFilled) {
  // Обновляем статус на "completed"
  await axios.patch(`/api/evaluation-sessions/${sessionId}/`, {
    status: 'completed'
  });
  alert(`✅ Успешно сохранено ${savedCount} оценок! Оценка завершена.`);
}
```

**Логика:**
- ✅ Проверка: заполнены ли все критерии
- ✅ Если ДА → статус = "completed"
- ✅ Если НЕТ → статус остается "in_progress"
- ✅ Информативное сообщение пользователю

└──────────┴──────────┴──────────┴──────────┘

### Роли и их описания:

#### 👑 Администратор системы
Полный доступ ко всем функциям системы: управление пользователями, настройка модели оценки, создание и редактирование продуктов, проведение оценок.

#### 🔍 Эксперт/Аудитор
Проведение оценок зрелости продуктов, верификация данных, просмотр результатов, генерация отчетов.

#### 📦 Владелец продукта
Управление своими продуктами, заполнение оценок, просмотр результатов своих продуктов.

#### 👀 Наблюдатель
Просмотр списка продуктов и результатов оценок. Доступ только для чтения.
