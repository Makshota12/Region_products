## 🚀 Как пользоваться

### 1. Просмотр продуктов
- Зайдите на главную страницу
- Кликните на любую карточку продукта
- Увидите детальную информацию

### 2. Создание оценки
- Перейдите в "📊 Сессии оценки"
- Нажмите "➕ Начать новую оценку"
- Выберите продукт
- Критерии создадутся автоматически!

### 3. Заполнение оценки
- Откройте сессию для оценки
- Увидите прогресс-бар
- Заполните оценки (1-10) для каждого критерия
- Можете добавить метрики и комментарии
- Прогресс обновляется автоматически

### 4. Просмотр результатов
- После заполнения нажмите "📊 Посмотреть результаты"
- Увидите:
  - Общий индекс зрелости
  - Уровень зрелости (Превосходный/Высокий/Средний/Низкий/Критический)
  - Оценки по доменам
  - Три типа графиков
  - Процент достижения максимума
- Скачайте PDF-паспорт
