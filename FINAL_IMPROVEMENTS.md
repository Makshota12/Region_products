## 🔄 Автоматизация

### При создании сессии оценки:
1. Автоматически создаются AssignedCriterion для всех критериев
2. Критерии назначаются создателю сессии
3. Готовы к заполнению

### При регистрации пользователя:
1. Автоматически создается Profile
2. Назначается роль "observer"
3. Привязывается к User

## 🎯 Функции показа/скрытия пароля

### Login.js
```javascript
const [showPassword, setShowPassword] = useState(false);

<button
  type="button"
  className="password-toggle"
  onClick={() => setShowPassword(!showPassword)}
>
  {showPassword ? '👁️' : '👁️‍🗨️'}
</button>
```
